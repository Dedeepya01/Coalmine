# Coalmine
# Coal Mine Productivity and Safety Management System (2024)

## Description
A Python desktop application to monitor coal mine production and safety incidents, providing secure data management and real-time monitoring.

## Features
- Secure login-based interface for authorized access
- Manage and store production and safety data using CSV files
- Interactive GUI using Tkinter for efficient data entry
- Visualize production trends using Matplotlib for analysis
- Emergency alert and safety incident notifications

## Core Concepts
GUI Development, File I/O, Data Structures, Data Visualization, Python Programming

## Technologies Used
Python, Tkinter, Matplotlib, CSV

## How to Run
1. Open **`coal_mine_system`** in Jupyter Notebook.  
2. Ensure all required libraries (Tkinter, Matplotlib, CSV) are installed.  
3. Run all cells in the notebook.  
4. An interactive GUI will open, allowing you to enter data, submit records, visualize production graphs, and trigger      emergency alerts.
